# Journal AI

Note ici tes interactions clés avec l'IA pendant l'exercice.

**Format libre** - pas de template imposé. L'important c'est qu'on puisse comprendre :
- Quels prompts ont bien marché (et pourquoi)
- Ce que tu as dû corriger/itérer
- Ce que tu as appris

Tu peux créer un fichier par exercice ou tout mettre ici, comme tu préfères.
